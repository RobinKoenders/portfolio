Ik heb de module Embedded-0 in klas 2 afgerond. Ik vond het een heel leuke eerste introductie naar de wereld van de Microbit. Het was naar mijn mening een redelijk makkelijke module en vond eigenlijk alleen de eindopdracht uitdagend. Ik vond het heel leuk om een keer een fysiek resultaat van mijn code te zien, omdat het voor mij veel meer belonend voelt dan een stukje tekst dat op het schermt verschijnt. Daarom denk ik erover om dit jaar nog een microbit of Arduino module te doen. Ik heb voor deze module 5 opdrachten en de eindopdracht "weerstation" gemaakt. Je kunt de videos hiervan vinden in mijn google drive folder.
